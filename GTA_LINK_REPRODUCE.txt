## â–¶ï¸ Commands Used

### 1. Tracklet Generation

```bash
python generate_tracklets.py \
  --model_path reid_checkpoints/sports_model.pth.tar-60 \
  --data_path dataset \
  --pred_dir submit \
  --tracker intermediate.txt
```

### 2. Tracklet Refinement (GTA-Link)

```bash
python refine_tracklets.py \
  --input_dir submit \
  --output_file submit/tracking_results.txt \
  --use_split \
  --min_len 50 \
  --eps 0.65 \
  --min_samples 15 \
  --max_k 2 \
  --use_connect \
  --spatial_factor 1.5 \
  --merge_dist_thres 0.35
```

---

## ğŸ§© Input Format

- `dataset/<video_id>/img1/`: Folder containing extracted frames as `000001.jpg`, `000002.jpg`, etc.
